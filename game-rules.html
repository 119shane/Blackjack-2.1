Blackjack 2.0 by Shane

GAME RULES

  • You need to obtain exactly the value "21" in the sum to win
  • Clicking "START GAME" button you`ll withdraw two cards, and sum is shown
  • You can withdraw another card if previous sum is less than 21
  • If sum gets exactly 21 you`ll win 10$ chips
  • If sum gets more than 21 you`ll lose 10$ chips
  • When chips reach 0 you`ll lose the game, if you win 150$ chips...(see you in version 2.1)
